Instructions


  1. Uninstall the original game.
  2. Download the .apk file
  3. Run the file and confirm install
  4. Open the new version of the game
  5. Join a game.
  6. Click the Krnl logo, which now appears at the top middle of the game
  7. Paste a script and execute it
